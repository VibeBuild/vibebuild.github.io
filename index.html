Conclusão

O código atingiu seu objetivo. É notável a praticidade para criação de scripts mais simples usando o vibe coding. Acredito ser muito útil para problemas simples, grades e demorados, mas pode ser um tiro no pé para questões mais complexas como algoritmos matemáticos e sistemas críticos. O uso da IA, no meu caso, foi de grande ajuda pois escreveu muitas linhas em menos de um dia. Entretanto, acredito que para quem não tem muita noção do que está fazendo, nem das ferramentas que usa, o caso é diferente, como aconteceu no início quando a ideia original era usar makefile. Sua aplicação é bem útil, no caso usei para compilar de uma vez diversos códigos de programação competitiva. A minha opinião em relação ao projeto é positiva, mas vejo uma complexidade de levar isso além daqui. O uso de shell script foi interessante por causa da interabilidade com o terminal linux. Apesar disso, não acredito que essa forma nem de longe chegue a ser a melhor forma séria, profissional e viável de se desenvolver um software. No meio do caminho, parecia que seria bem sólido, de certa forma ainda está, mas com o passar do desenvolvimento provavelmente ficará inconsistente.

A IA ameaça o mercado de trabalho da programação?

- Para quem trabalha com soluções simples, como criar um blog estático ou scripts de automação menores, talvez sim.
- Diria que a IA não teria capacidade nesses próximos anos para substituir igualmente um humano no seu trabalho em determinados setores da programação.
- Designs mais complexos, Backends com alta necessidade de desempenho, programação em embarcados de mais baixo nível, algoritmos matemáticos com alto grau de dificuldade que precisam de complexidades Big-O mínimos e sistemas que gerenciam dados sensíveis são exemplos de coisas que nem podemos pensar em atribuir o trabalho às IA's.

VibeBuild é uma ferramenta simples para automação de compilação.

Ela nasceu da minha necessidade de compilar pequenos trechos de código de forma rápida e prática, sem precisar configurar Makefiles ou outras ferramentas complexas.

Juntando isso com uma pesquisa escolar sobre vibe coding, nasceu o VibeBuild.

✨ Funcionalidades

  • 🛠️ Compilação rápida para pequenos códigos.
  • 🚀 Suporte a múltiplas linguagens.
  • 📜 Geração de log detalhado.
  • 🔥 Executar após compilar (--run ou -r).
  • 🏗️ Alterar de GCC pra CLANG (--clang).
  • 📁 Definir o diretório de saída (-d).

🚀 Como Usar

  1. Clone o repositório: 
    git clone https://github.com/VibeBuild/vibebuild.github.io
    cd vibebuild
  2. Deixe o instalador executável: 
    chmod +x installer
  3. Execute para instalar: 
    ./installer
  4. Execute o programa: 
    vibe
